При генерации/апдейте .edmx файла, может генерировать не полный список аргументов.
﻿﻿1. Если существуют ограниченные права доступа к процедурам, тогда необходимо фреймворку давать наивысший доступ (sa пользователя).
2. При генерации списков аргументов процедуры, Entity берет последний SELECT в процедуре и на основе его генерирует список аргументов. Если последний SELECT делает выборку из временной таблицы 
(SELECT * FROM #TempTB), аргументы сгенерированны не будут!.
