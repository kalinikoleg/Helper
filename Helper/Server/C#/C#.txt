--Методы расширения
это статический метод, который можно вызывать на уровне экземпляра,
а не на самом классе.

	public static class StringConversation
    {
        public static double ToDoube(this string s)
        {
            return Double.Parse(s);
        }
    }

	double ds = "1".ToDoube();
будет вызываться для типа, который указан в (this string s). 
И метод, и класс - являются статическими.

--Делегат Func
public delegate TR Func<T0,T1,T2, TR>(T0 a0, T1 a1 ,T2 a2)
T0,T1,T2 - ссылаются на входные параметры, переданные методу.
TR - возвращаемый тип данных, он в конце всегда.
